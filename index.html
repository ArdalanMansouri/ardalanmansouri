About

I am a computational biologist and data scientist specialising in software solutions for medical, pharmacological, and biotech research settings. My work focuses on building transparent, reproducible, and open-source Python tools that bridge the gap between raw instrument output and actionable biological insights.

My projects span high-content imaging analysis, flow cytometry visualisation, cytotoxicity assay pipelines, and single-cell data workflows — all designed to replace ad-hoc spreadsheet analyses with well-documented, notebook-friendly packages. Research funded in part by the European Union's Horizon 2020 programme (proEVLifeCycle, grant No 860303).

flowcytometry-graph-gen

An open-source Python alternative to black-box commercial tools (e.g., FlowJo) for processing and visualising flow cytometry data. Built on Plotly, it provides full transparency into every normalisation step and enables extensive customisation of publication-quality interactive figures.

cytotoxicity-analyzer

cytotox — a complete, code-first pipeline for cytotoxicity assay analysis. Classifies single cells into Viable, Apoptosis, Late Apoptosis, and Necrosis populations using Caspase 3/7 and PI fluorescence intensity thresholds, with interactive Plotly scatter plots and HTML export. Live notebooks available at ardalanmansouri.github.io/cytotoxicity-analyzer .

Python Cytotoxicity Plotly Cell Biology
Python MIT License

cell-growth-analyzer

cellgrowth — a lightweight, Jupyter-friendly Python package for reproducible analysis and interactive visualisation of cell growth data exported from high-content imaging plate readers (e.g., PerkinElmer Operetta/Opera). Converts raw timepoint exports into statistically annotated grouped tables and interactive Plotly line graphs with error-bar toggles.

Python Cell Growth HCI Plate Reader
Python MIT License

high-throughput-compound-screening

End-to-end Python library for high-throughput compound screening (HTS) data analysis in drug discovery. Ingests raw plate-reader output, applies plate-to-plate normalisation, performs SD-based hit classification (inhibitors / inducers), and renders well-layout heatmaps. Supports LOPAC and Prestwick compound libraries.

Python Drug Discovery HTS Plate Analysis
